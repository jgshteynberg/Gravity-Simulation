# Three-Body-Problem

An interactive browser-based 2D gravity sandbox for experimenting with N-body motion.

## Features

- Drag bodies around the canvas to change their starting positions.
- Adjust each body's mass and starting velocity with sliders.
- Add or remove bodies from the simulation.
- Pause, play, and reset the simulation while trails visualize recent motion.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server, for example:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
